## üßÆ Experiment 1: Using Formulas in Excel  

**Aim**  
To perform basic and advanced calculations using Excel formulas.  

**Steps**  
1. Open Microsoft Excel and create a new workbook.  
2. Enter the following sample data:  
   | A   | B   |  
   |-----|-----|  
   | 10  | 20  |  
   | 30  | 40  |  
   | 50  | 60  |  
3. In cell C1, type `=A1+B1` and press Enter (Addition).  
4. Drag the Fill Handle down to apply it to other rows.  
5. Try other basic formulas:  
   - Subtraction ‚Üí `=A1-B1`  
   - Multiplication ‚Üí `=A1*B1`  
   - Division ‚Üí `=A1/B1`  
   - Average ‚Üí `=AVERAGE(A1:B3)`  
   - Sum ‚Üí `=SUM(A1:B3)`  
6. Try an advanced function:  
   - `=IF(A1>20,"High","Low")`  

**Result**  
Successfully used Excel formulas to calculate and analyze data.

## üßä Experiment 2: Freeze Panes  

**Aim**  
To learn how to freeze rows and columns for easy data viewing.  

**Steps**  
1. Open a workbook with large data (more than one screen).  
2. Go to the View Tab ‚Üí Freeze Panes.  
3. Choose one of the following:  
   - Freeze Top Row ‚Äì Keeps the first row visible.  
   - Freeze First Column ‚Äì Keeps the first column visible.  
   - Freeze Panes ‚Äì Freezes both rows and columns up to the selected cell.  
4. Scroll the sheet and observe that the frozen part remains visible.  
5. To remove freezing, click View ‚Üí Unfreeze Panes.  

**Result**  
Successfully used Freeze Panes to keep headings visible while scrolling.

## üé® Experiment 4: Conditional Formatting  

**Aim**  
To apply conditional formatting for highlighting data based on conditions.  

**Steps**  
1. Open an Excel sheet with numeric or text data.  
2. Select the data range to format.  
3. Go to Home ‚Üí Conditional Formatting.  
4. Choose one of the following rules:  
   - Highlight Cells Rules ‚Üí Greater Than, Less Than, Between.  
5. Example:  
   - Select marks column ‚Üí Conditional Formatting ‚Üí Highlight Cells ‚Üí Greater Than ‚Üí Type 50 ‚Üí Choose color.  
6. Observe cells that meet the condition change color.  

**Result**  
Applied and verified conditional formatting for different conditions.

## üîê Experiment 5: Document Encryption in Excel  

**Aim**  
To secure an Excel workbook by adding a password.  

**Steps**  
1. Open the Excel workbook you want to protect.  
2. Click File ‚Üí Info ‚Üí Protect Workbook ‚Üí Encrypt with Password.  
3. Enter and confirm your password.  
4. Save and close the workbook.  
5. Reopen it ‚Äî Excel will prompt you to enter the password.  
6. To remove the password:  
   - Go to File ‚Üí Info ‚Üí Protect Workbook ‚Üí Encrypt with Password.  
   - Delete the existing password and click OK.  
   - Save again.  

**Result**  
Successfully applied and removed password protection for an Excel workbook.  

# üìä6 : Create a Pivot Table in Excel

## üß† Step-by-Step Instructions

### 1. Select Your Data  
Click anywhere inside your table (Excel will detect the range automatically).

---

### 2. Insert Pivot Table  
Go to the top menu:  
**Insert ‚Üí PivotTable**

In the dialog box:  
- Choose **Select a table or range**  
- Verify that your range is correct  
- Choose where you want the Pivot Table (**New Worksheet** recommended)  
- Click **OK**

---

### 3. Build Your Pivot Table  
You‚Äôll now see the **PivotTable Fields** pane on the right.

Drag and drop fields into these areas:

| Area | Field |
|------|--------|
| **Rows** | Region |
| **Columns** | Product |
| **Values** | Total Sales *(set to ‚ÄúSum of Total Sales‚Äù)* |
| **Filters (optional)** | Salesperson |

---

### 4. Format Your Pivot Table  
Click inside the Pivot Table ‚Üí Go to:  
**PivotTable Analyze ‚Üí Design**

Then:  
- Choose a style you like  
- Enable or disable totals as needed
